WindowsDoctor USB 入口

這支 USB 已整合 2 個功能：正常 Windows 工具包 + 不能開機時的 WinPE 救援。

🖥️

電腦可以正常開機

  1. 插入 USB。
  2. 打開 G:\WindowsDoctor-PortableUSB-GUI-READY-20260503。
  3. 要看圖形介面：雙擊 Start-WindowsDoctor-GUI-Ready.cmd。
  4. 這會先同步到本機快取，再開啟 http://localhost:3000。
  5. 只要安全文字選單：雙擊 Start-WindowsDoctor-Portable.cmd。
  6. 依畫面操作診斷、查詢錯誤、匯入 NotebookLM JSON。
🛟

電腦不能正常開機

  1. 關機後插入 USB。
  2. 開機時按 Boot Menu 鍵，例如 F12、F11、Esc。
  3. 選擇這支 USB 開機。
  4. 進入 WinPE 後使用 WindowsDoctor 文字選單。
安全規則：診斷與預覽可以直接使用；真正修復前需要看到 RUN 確認。
NotebookLM GUI 圖文說明：打開 G:\WINDOWSDOCTOR_NOTEBOOKLM_GUI_USER_GUIDE.html
