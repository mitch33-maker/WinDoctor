# VSS Provider Veto (0x80042306) 故障修復

## 症狀描述
當備份軟體或系統還原點建立失敗，並出現 `0x80042306` 錯誤時，通常代表磁碟區陰影複製服務 (VSS) 被第三方提供者或磁碟空間限制所「否決」(Veto)。

## 研判邏輯 (RAG Context)
此錯誤多與 **VSS Writers 狀態受損** 或 **Shadow Storage 空間不足** 有關。

## 修復方法
1. **重置服務狀態**：重新啟動 VSS 與關聯服務。
2. **清理陰影複製空間**：刪除過時或損壞的快照，並擴充最大可用空間。
3. **系統修復**：執行 SFC 確保 VSS 組件完整。
