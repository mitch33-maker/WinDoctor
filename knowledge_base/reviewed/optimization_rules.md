# 系統效能優化與維護

## 建議清單
1. **清理臨時檔案**：釋放磁碟空間並減少檔案系統索引負擔。
2. **磁碟重組與優化 (Trim)**：提升 SSD/HDD 讀寫效率。
3. **清理 DNS 快取**：解決網路連線遲緩問題。
