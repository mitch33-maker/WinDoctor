正常 Windows 使用

雙擊 Start-WindowsDoctor-GUI-Ready.cmd，打開圖形介面。

📚

NotebookLM 資料接入

在 GUI 裡選 JSON，按 匯入並重建。

🛟

不能開機救援

用這支 USB 開機，進入 WinPE 離線選單。

一、啟動 GUI

1

插入 USB，打開 G:\START_HERE.html。

2

進入 G:\WindowsDoctor-PortableUSB-GUI-READY-20260503。

3

雙擊 Start-WindowsDoctor-GUI-Ready.cmd。

4

瀏覽器會開啟 http://localhost:3000。

三、準備 NotebookLM JSON

1

範本在：G:\WindowsDoctor-PortableUSB-GUI-READY-20260503\WindowsDoctor\templates\NOTEBOOKLM_SOURCE_PACK_TEMPLATE.json

2

把 NotebookLM 整理好的錯誤、症狀、修復建議填進 JSON。

3

儲存成例如：G:\notebooklm-source-pack.json。

四、匯入資料

1

在 GUI 的 NotebookLM 區塊按 選取 JSON。

2

確認右邊「接入檢查」沒有紅色錯誤。

3

按 匯入並重建。

4

看到「匯入完成」代表資料已進入 WindowsDoctor normalized KB。

五、Google Pro 與 Enterprise API

Google Pro 會員可使用 NotebookLM 網頁功能與較高額度，適合用「匯出 / 整理 JSON」方式接入 WindowsDoctor。

直接 API 連線需要 NotebookLM Enterprise、Google Cloud Project、IAM 權限與 access token；一般 Pro 會員不會自動取得這些 API 權限。

六、指令備用方案

如果 GUI 不能用，可用這條指令匯入：

powershell -NoProfile -ExecutionPolicy RemoteSigned -File G:\WindowsDoctor-PortableUSB-GUI-READY-20260503\WindowsDoctor\scripts\Import-NotebookLMSourcePack.ps1 -Root G:\WindowsDoctor-PortableUSB-GUI-READY-20260503\WindowsDoctor -InputPath G:\notebooklm-source-pack.json -ReportPath G:\notebooklm-import.json -Json
