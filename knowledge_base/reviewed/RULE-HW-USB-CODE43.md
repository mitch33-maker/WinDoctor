# USB 裝置 Code 43 或裝置描述元失敗
- EventID/Code: CODE43_USB
- Trigger: ["Code 43", "USB device descriptor failed", "Unknown USB Device", "Device Descriptor Request Failed"]
- Script: "N/A"

## 分析細節
常見於 USB 裝置故障、供電不足、集線器異常或驅動快取錯誤。引導使用者換 USB 孔、移除集線器、完全關機斷電後再測試；若多台電腦皆失敗，優先判定硬體故障。
